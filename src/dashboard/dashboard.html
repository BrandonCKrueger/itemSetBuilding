Welcome!

Welcome to the Online Item Set Builder for League of Legends.

This application was build by Brandon Krueger for entry into Riot Game's API Challenge 2.0

In order create and share Item Sets you will need to create an account. If you don't feel like creating an account, you can still browse other Summoner's builds and download their Item Sets.

Browse the Champions list to get started!

Interested in helping, or just learning how it was made?

The application is broken into two seperate projects.

  1. Frontend Angular Application ( on GitHub)
  2. Backend Node Application ( on GitHub)
Visit either one of the github pages to learn more about how each eas created.

Feel free to contact me at brandon.c.krueger@gmail.com if you want some resources to learn the skills used to create these applications
